# Bot egg

### Supported languages
- Node.js
- Python 2 and 3
- Java 11 (OpenJDK)
- .NET
- C#
- Lua 5.3
- Golang
- Elixir

### Downloading the egg
To download the egg, click on the .json file in this directory. After that, right-click on the "Raw" button and select "Save link as" or something like that.
